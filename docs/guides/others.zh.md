# 杂项

## 贡献

欢迎贡献！你可以通过以下方式贡献。

- 创建 Issue - 提议新功能。报告 bug。
- Pull Request - 修复 bug 和拼写错误。重构代码。
- 创建第三方中间件 - 参考下方说明。
- 分享 - 在博客、X(Twitter) 等平台上分享你的想法。
- 创建你的应用程序 - 请尝试使用 Hono。

有关更多详细信息，请参阅 [贡献指南](https://github.com/honojs/hono/blob/main/docs/CONTRIBUTING.md)。
